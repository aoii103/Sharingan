## 环境安装

首先保证您已经安装了```python3.8```及以上版本,然后依次运行如下命令。

```sh
git clone https://github.com/aoii103/Sharingan.git

cd Sharingan

python3 -m pip install -r requirements.txt
```

```sh
python3 worker.py blue
```
